# Security Policy

## Supported Versions

SUSHITAP is a small personal app with no formal release branches. Only the latest version published on the `main` branch and on the [Releases](https://github.com/raamonsiu/SUSHITAP/releases) page is supported with security fixes.

## Reporting a Vulnerability

SUSHITAP has no backend, no accounts, and no network calls: all data is stored locally on the device with AsyncStorage. Even so, if you find a security issue (for example, something related to how the app handles stored data, or a vulnerable dependency), please report it privately instead of opening a public issue.

Send a report to **ramonjunior04@gmail.com** with a description of the issue and, if possible, steps to reproduce it. You should receive a response within a few days. Once the issue is confirmed, a fix will be released as soon as possible and credit will be given to the reporter unless anonymity is requested.
